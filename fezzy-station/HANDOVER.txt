GET /health
→ Returns: "OK" (200)
→ Used by Render for health monitoring

┌─────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                 │
└─────────────────────────────────────────────────────────────────┘

Problem: Site won't load
Fix:     Check Render dashboard for deployment errors
         Verify requirements.txt has all dependencies

Problem: Search doesn't work
Fix:     Check browser console for errors
         Verify external sites haven't changed URLs

Problem: IP lookup fails
Fix:     Ensure Flask backend is running
         Check /api/ip endpoint in browser

Problem: Adult portal won't expand
Fix:     Check JavaScript console
         Verify THE_MAIN_ONE.html loaded correctly

Problem: Git push fails
Fix:     Use Personal Access Token, not password
         Verify remote URL: git remote -v
